# Windows客户端
这是一个使用Python制作的，基于PyQt5和PyQt Fluent Widgets(https://github.com/zhiyiYo/PyQt-Fluent-Widgets)
基于PyQt5开发，需要使用者拥有Python环境，支持Windows 8.1及以上版本系统
# Android客户端
基于WebWiew开发，支持Android 9及以上版本系统，尚处于早期开发阶段，功能较简陋
