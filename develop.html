插件开发文档

当前为测试版，可能有较大变动，请注意
插件为文件夹形式
文件夹下有addon.json文件，格式为，不可缺项，如果网页不允许__init__.py（如githubpages）可改名为init.py（程序自动转换）
{"id": "英文小写","name": "随意，建议中文，会在程序中展示","version": "如1.0.0","file": ["必有__init__.py"],"lib": [可留空]}
__init__.py开头需要：
import sys, os
sys.path = [os.path.dirname(sys.argv[0])] + sys.path
from source.custom import *
os.chdir(__file__)
其他运行库自行导入即可
页面组件为class AddonTab(BasicTab):
AddonTab需要配置如下选项：
self.setIcon(FIF.插件图标)
