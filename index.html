Windows客户端
立刻下载 更新日志
基于PyQt5开发，需要使用者拥有Python环境，支持Windows 7及以上版本系统
Python 3.11.4
在安装前请注意一定要勾选下方“Add Python 3.X to PATH”选项，点击“Install Now”即可安装Python
运行安装器，首先需要点击“安装 zb小程序 运行库”并等待安装
在选择安装目录后，点击“立刻安装 zb小程序 最新版本”，等待安装
安装完毕后，会自动打开zb小程序，在设置页面添加快捷方式并添加开机自启动，即可开始使用zb小程序
如果遇到Bug，请在Github提出Issue
