使用教程
立刻下载
!!!先看完再安装!!!
本软件基于Python开发，需要使用者拥有Python环境，点击下方链接下载并安装，注意系统版本
Python 3.11.3 Python 3.10.11（Windows 8.1/10/11） Python 3.8.10（Windows 7）
在安装前请注意一定要勾选下方“Add Python 3.X to PATH”选项，点击“Install Now”即可安装Python
将下载到的zb小程序安装器移动到其他文件夹，zb小程序将安装到安装器同一文件夹下
运行安装器，首先需要点击“安装 zb小程序 运行库”并等待进度条加载
点击“立刻安装 zb小程序 for XXX”，等待进度条加载，程序可能会短暂卡顿，取决于网络状态
安装完毕后，会自动打开zb小程序，可通过下方检测更新按钮打开安装器以安装最新版本，也可以将贴边窗口添加至开机自启动
安装器目录下的“main.pyw”文件即为zb小程序主程序，请自行添加桌面快捷方式
安装的版本默认为Myself版本，如需更换版本请进入设置，”一键整理+清理“功能会整理桌面和微信文件等至其他目录，请在设置里自行修改目录
